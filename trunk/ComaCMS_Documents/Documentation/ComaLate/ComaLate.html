method SetReplacement [line 177]

	

    

    

		void SetReplacement(
string
$Name, [string/array
$Value = ''])
    

    


	
		Adds or changes a replacement (if the replacement already exists) in one of the local replacement arrays

A single replacement looks like this:  {NameOfTheReplacement}
A loop replacement looks like this:
<NameOfTheReplacement:loop>    {ASingleSubReplacement}{AnotherSingleSubReplacement}{ASingleSubReplacement}      <ALoopSubReplacement:loop>        {ASingleSubReplacement}{AnotherSingleSubReplacement}{ASingleSubReplacement}      </ALoopSubReplacement>  </NameOfTheReplacement>





Tags:





  

    return:  Sets a replacement
  









	
        
Parameters:

    

    

          

        string  
        $Name  
        The name of the replacement
      

          

        string/array  
        $Value  
        The value of the replacement. Can be a [string] for a single Replacement or an [array] for a "loop" replacement
