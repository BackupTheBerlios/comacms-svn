Features of the "Textcompiler"

Default Fomrmattings

Bold

Input:

**TEXT**

Output:

<strong>TEXT</strong>

Underline

Input:

__TEXT__

Output:

<u>TEXT</u>

Italic

Input:

//TEXT//

Output:

<em>TEXT</em>

Center

Input:

<center>TEXT<center>

Output:

<div class="center">TEXT</div>

Headers

H2 (highest)

Input:

== TEXT ==

Output:

<h2>TEXT</h2>

H3 (middle)

Input:

=== TEXT ===

Output:

<h3>TEXT</h3>

H4 (lowest)

Input:

==== TEXT ====

Output:

<h4>TEXT</h4>

Links

Link

Input:

www.example.com

Output:

<a href="http://www.example.com" class="link_extern">
www.example.com</a>

Link

Input:

[[www.example.com|Example]]

Output:

<a href="http://www.example.com" class="link_extern">Example</a>

Link(intern)

Input:

[[intern|Example]]

Output:

<a href="index.php?page=intern" class="link_intern">Example</a>

Email

Input:

<name@example.com>

Output:

<a href="mailto:name [at] example [dot] com" class="link_email">
name [at] example [dot] com</a>

Image

List(ordered)

List(unordered)
